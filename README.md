TURF BOOKING SYSTEM
A Java console application to manage sports turf bookings. Users can view turfs, book slots, cancel bookings, and add reviews. Vendors can manage their turfs, add slots. Admins can manage users, vendors, and turfs.
Features :
- View turfs and available slots
- Book and cancel slots
- Add and view turf reviews
- Vendor turf and slot management
- Admin management of users, vendors, and turfs
Key Highlights
- Strong Core Java skills: Classes, Objects, Inheritance, Polymorphism
- Collections and data management with Lists
- Console-based user interaction
- Multi-role management (User, Vendor, Admin)
- Booking and slot management logic
